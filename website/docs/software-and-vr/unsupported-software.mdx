# Unsupported Software

Software that is not listed as [Fully Supported](../fully-supported-software) or [Recommended Free Software](../recommended-free-software) is not officially sanctioned or supported by UPrep. Please ensure you follow data safety and privacy guidelines when using it, including not putting student data in unapproved software. Faculty and Staff may request support for additional software by following the process on the Academic Tech for Faculty site (linked from Veracross).
