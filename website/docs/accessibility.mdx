# Accessibility Tools for Students and Faculty

Every major platform—Apple, Google, Microsoft, and more—includes powerful
built-in accessibility features for vision, hearing, motor, and learning needs.

Below are some of the most useful accessibility tools available on school
devices and personal technology.

## Apple Devices (macOS and iOS)

Apple includes accessibility tools in every Mac, iPad, and iPhone:

- VoiceOver — Full-featured screen reader that describes everything happening on
  screen.

- Zoom and Display Settings — Screen magnification and text size adjustments for
  visual clarity.

- Speak Selection / Speak Screen — Reads selected text or full pages aloud.

- Live Captions — Automatically captions spoken dialogue and audio.

- Dictation and Voice Control — Hands-free text entry and navigation using voice
  commands.

- Background Sounds — Plays ambient audio (like rain or white noise) to aid
  focus or calm.

- Color Filters and Reduce Motion — Customizes visual display for color
  blindness or sensory needs.

Learn more at [Apple Accessibility](https://www.apple.com/accessibility/).

## ChromeOS and Chromebooks

Google’s ChromeOS includes accessibility features ideal for classroom and 1:1
environments:

- Select-to-Speak — Highlights and reads selected text aloud.

- Screen Reader (ChromeVox) — Full screen reading for blind and low-vision
  users.

- Dictation — Converts speech to text in any input field.

- Magnifier and High Contrast Mode — Helps with visual accessibility.

- Live Caption (Chrome browser) — Provides automatic captions for videos and
  audio in real time.

- Keyboard Shortcuts & On-Screen Keyboard — Simplifies navigation and input.

See
[Chromebook Accessibility Features](https://edu.google.com/intl/ALL_us/our-values/accessibility/chromebooks-accessibility/).

## Android Devices

Android offers a wide range of tools through system settings and Google apps:

- TalkBack — Screen reader that describes on-screen items and navigation.

- Select-to-Speak — Reads text aloud after tapping it.

- Live Caption — Instant captions for videos, podcasts, and voice messages.

- Voice Access — Complete control of your device with spoken commands.

- Magnification and Font Size Controls — Flexible visual aids.

Details at
[Android Accessibility](https://support.google.com/accessibility/android).

## Windows and Microsoft Tools

Many students and teachers use Microsoft tools daily; the built-in accessibility
options can make a big difference:

- Immersive Reader (in Word, OneNote, Edge, and Teams) — Reads text aloud,
  highlights words, and adjusts spacing or background color.

- Windows Narrator — Screen reader for text and interface elements.

- Magnifier, High Contrast, and Color Filters — Visual enhancements for clarity.

- Speech Recognition — Hands-free navigation and typing.

- Focus Assist — Reduces on-screen distractions.

Learn more at
[Microsoft Accessibility](https://www.microsoft.com/accessibility).

## Helpful Chrome Extensions

Browse the
[Chrome Web Store Accessibility Collection](https://chromewebstore.google.com/category/extensions/make_chrome_yours/accessibility).

## Built-In Tools Across Platforms

Most major tools UPrep already uses—Google Docs, Slides, Sheets, Canvas, and
Schoology—include built-in accessibility features like keyboard navigation,
text-to-speech compatibility, and alt-text for images.

## Faculty Guidance

Faculty are encouraged to:

- Provide digital copies of materials for use with assistive tools.

- Add alt text to images in slides and documents.

- Use clear headings and structured text for screen readers.

- Enable captions on videos shared in class.
