## Question

How do I make myself an admin on a course?

## Answer

You'll need to ask a current admin (generally the teacher) to make you an admin.
From the course, they can go to Members, click the gear icon next to your name,
and choose Make Admin. See the
[Schoology Help Page](https://uc.powerschool-docs.com/en/schoology/latest/assign-section-override-roles-instructors)
for details.

If there are no course admins still working at UPrep,
[reach out to June](mailto:jpeters@universityprep.org) to get yourself added.
