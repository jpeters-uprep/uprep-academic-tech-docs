## Question

How do I make myself an admin on a course?

## Answer

You'll need to ask a current admin (generally the teacher) to make you an admin.
They can add you and make you an admin using
[this information from SPS](https://www.seattleschools.org/wp-content/uploads/2021/07/HowToAddACoteacher.pdf).

If there are no course admins still working at UPrep,
[reach out to June](mailto:jpeters@universitypre.org) to get yourself added.
