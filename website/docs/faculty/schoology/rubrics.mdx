Rubrics let you score student work against a set of criteria, and can save you
time if you use similar criteria across multiple assignments.

A few reasons to consider using rubrics in Schoology:

- You can use a standard rubric and still add personal comments to an individual
  student's submission.
- A rubric can be reused across multiple assignments, courses, or shared with
  colleagues.
- Rubrics work even for offline or physically-submitted work — students don't
  need to submit a file through Schoology for you to grade them with a rubric.

Rubrics are optional at UPrep, but many teachers find them useful for
assessments with several graded criteria.

You can create a rubric directly from Grade Setup, or while creating an
assignment, assessment, discussion, or quiz. Rubrics you build from an
assignment can be saved back to Grade Setup for reuse, and rubrics saved to your
Resources can be copied into any course you manage.
